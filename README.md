# Smart Event Dashboard

**Web Dev II – Lab Assignment 1**

## Introduction

This is my Lab Assignment 1 for Web Development II. In this assignment, I have created a **Smart Event Dashboard** using basic **HTML, CSS, and JavaScript** without using any external libraries or frameworks.

The main objective of this project is to understand how **DOM manipulation** works in JavaScript. In this dashboard, users can create event cards dynamically, view them, and interact with them using different JavaScript events.

Through this assignment, I practiced:

* Creating and removing HTML elements using JavaScript
* Handling user events like click and input
* Using DOM traversal methods
* Understanding the difference between `innerText`, `innerHTML`, and `textContent`
* Basic debugging using browser DevTools

This project helped me improve my understanding of how JavaScript interacts with HTML elements and how dynamic web pages are built using pure JavaScript.
